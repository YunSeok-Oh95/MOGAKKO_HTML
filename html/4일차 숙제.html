HTML5

문서
토론

HTML5는 HTML의 완전한 5번째 버전으로 월드 와이드 웹 (World Wide Web)의 핵심 마크업 언어이다.

2004년 7월 Web Hypertext Application Technology Working Group(WHATWG)에서 웹 애플리케이션 1.0이라는 이름으로 세부 명세 작업을 시작하였다.

W3C는 2014년 10월 28일 HTML5 표준안을 확정했다고 발표했다.

이후 2016년 11월 1일 HTML5의 마이너 업데이트 버전인 HTML5.1 표준안을 확정, 2017년 12월 14일 HTML5.2 표준안을 확정했다.

HTML5.3 표준안은 현재 작업 초안 단계로 진행 중이다.



HTML의 발전 역사

HTML은 1991년부터 2007년까지 HTML1.0부터 현재의 HTML5.0까지 시간이 지나면서 발전해왔다.

  • 1991년 - HTML 1.0
  • 1995년 - HTML 2.0
  • 1997년 - HTML 3.2
  • 1999년 - HTML 4.01
  • 2007년 - HTML 5
